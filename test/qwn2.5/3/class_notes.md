# Sistemas bioinspirados: el juego de la vida

## Optimización con métodos bioinspirados
### Algoritmos Evolutivos
- **Concepto:** Mimetiza los procesos evolutivos naturales para resolver problemas complejos.
- **Key Takeaway:** Utiliza selección, cruce y mutación para mejorar soluciones iterativamente.

### Colonias de hormigas
- **Concepto:** Basado en la búsqueda eficiente de recursos por las hormigas.
- **Key Takeaway:** Utiliza reglas simples locales para resolver problemas complejos globalmente.

### Inteligencia de enjambres
- **Concepto:** Modelado del comportamiento colectivo y cooperativo de organismos sociales.
- **Key Takeaway:** Maximiza la eficiencia a través de interacciones locales entre individuos.

## Introducción a las redes neuronales: el modelo de la neurona de los mamíferos
### Perceptrones y backpropagation
- **Concepto:** Un paradigma de aprendizaje supervisado que utiliza gradientes para ajustar parámetros.
- **Key Takeaway:** Propagación hacia atrás es crucial para ajustar pesos en capas ocultas.

## Aplicación de redes neuronales a datos tabulares
### Regresión
- **Concepto:** Predicción continua de una variable dependiente.
- **Key Takeaway:** Útil para pronosticar valores numéricos basados en características.

### Series de tiempo
- **Concepto:** Prevención futuros valores basándose en datos pasados.
- **Key Takeaway:** Ideal para predecir tendencias y patrones en secuencias temporales.

### Clasificación
- **Concepto:** Categorización de datos en diferentes clases.
- **Key Takeaway:** Útil para problemas donde se necesita asignar un dato a una clase.

## Aprendizaje profundo y frameworks de trabajo
### Aumentación de datos
- **Concepto:** Generación artificial de muestras adicionales mediante técnicas como rotaciones, zooms, etc.
- **Key Takeaway:** Mejora la robustez del modelo con más datos variados.

### Redes neuronales convolucionales y aplicaciones en imágenes
- **Concepto:** Diseñadas para manejar entradas de datos multidimensionales, particularmente imágenes.
- **Key Takeaway:** Máximiza la capacidad del modelo para entender patrones espaciales.

### Aprendizaje por refuerzo
- **Concepto:** Forma en que un agente aprende a realizar acciones óptimas maximizando recompensas.
- **Key Takeaway:** Ideal para problemas donde se requiere tomar decisiones secuenciales basadas en feedback.

### Aprendizaje adversarial
- **Concepto:** Involucra dos redes neuronales que se entrenan mutuamente, una generando y otra distinguiendo.
- **Key Takeaway:** Útil para problemas donde la creación de datos falsos es crucial (ej. GANs).

### Difusión estable (Stable Diffusion)
- **Concepto:** Un algoritmo para generar imágenes a partir de texto, mediante un proceso iterativo.
- **Key Takeaway:** Permite la creación de contenido visual basado en descripciones textuales.

### Redes neuronales recurrentes y transformers
- **Concepto:** Diseñadas para manejar secuencias temporales y datos desordendados.
- **Key Takeaway:** Útiles para problemas que involucran información temporal, como la traducción de idiomas.
